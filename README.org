* vscode-org-mode

A Visual Studio Code extension providing [[https://orgmode.org/features.html][org-mode]] language support, powered by [[https://github.com/lsp-org-mode/lsp-org-mode][lsp-org-mode]].

** Syntax Highlighting

Full syntax highlighting support for org-mode files.

* Installation

** Prerequisites

Install the [[https://github.com/lsp-org-mode/lsp-org-mode][lsp-org-mode]] language server:

#+begin_src shell
git clone https://github.com/lsp-org-mode/lsp-org-mode.git
cd lsp-org-mode
ln -s "$(pwd)/bin/lsp-org-mode" ~/.local/bin/
#+end_src

Ensure that =~/.local/bin= is in your =PATH=.

** Building from Source

Clone and build the extension:

#+begin_src shell
git clone https://github.com/lsp-org-mode/vscode-org-mode.git
cd vscode-org-mode
pnpm install
pnpm build
#+end_src

* Development

1. Open the project in VS Code:

   #+begin_src shell
   cd vscode-org-mode
   code .
   #+end_src

2. Press =F5= to launch a development instance with the extension loaded.

3. Open any =.org= file to test the extension.
